# 🌍 How Well Do You Know Africa?

> *A quiz that challenges the world's knowledge of the most misunderstood continent.*

---

## 💡 About The Project

Most people know very little about Africa beyond the stereotypes. This quiz changes that — covering Ghanaian history, African geography, culture, and technology in a fun, interactive way.

Built with Python as one of my first projects as a self-taught developer. No libraries. No frameworks. Just pure Python and curiosity. 🐍

---

## 🎮 Features

- 4 categories to choose from
- 25 questions total across all categories
- Questions and options shuffle randomly every game
- Instant feedback after every answer
- Score and performance message at the end
- Play again without restarting the program

## 🚀 How To Run

1. Make sure Python 3 is installed
2. Clone this repository:
   ```bash
   git clone https://github.com/Suzzane-Nhyira/how-well-do-you-know-africa.git
   ```
3. Navigate into the folder:
   ```bash
   cd how-well-do-you-know-africa
   ```
4. Run the game:
   ```bash
   python quiz_game.py
   ```

No installations needed. Just Python. ✅
